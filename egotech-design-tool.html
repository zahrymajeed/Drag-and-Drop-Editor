How to Use FrameForge

EgoTECH Design Tool
⚡ Quick Workflow
1. Choose canvas size→ 2. Drag a component→ 3. Move & resize→ 4. Double-click to edit text→ 5. Export HTML
📋 Step by Step
1

Choose canvas size

Use the dropdown in the toolbar — Desktop, Tablet, Mobile, or Social OG. Canvas auto-zooms to fit.

2

Drag components onto canvas

Grab any card from the left sidebar and drop it anywhere on the canvas. It lands exactly where you release.

3

Select & move elements

Click an element to select it (purple outline). Then drag to reposition it freely anywhere on the canvas.

4

Resize with handles

8 white handles appear on selection. Drag corners or edges to resize. Corners = diagonal, edges = one axis.

5

Double-click to edit text

Double-click any element to edit its text inline. Click outside when done to save your changes.

6

Properties panel (right)

Select an element to control X, Y, width, height, opacity. Use action buttons to duplicate or delete.

7

Layers tab

Switch to Layers to see all elements. Click to select. Toggle 👁 to show/hide. Use Front/Back to reorder.

8

Export HTML

Click the green Export HTML button. Downloads a standalone .html file ready for browsers or code editors.

✨ New Components (Your RM's Requests)
📌 Sticky Note

Drag onto canvas → a popup appears to type your note and choose a colour. Great for annotations and team reminders.

▶ YouTube Embed

Drag onto canvas → paste any YouTube URL or video ID. The video embeds live and plays directly inside your design.

💻 Code Block

Drag onto canvas → choose language, paste code. Renders with dark-theme syntax highlighting and a macOS-style title bar.

🧠 Code + Explanation

Same as Code Block but adds an explanation panel on the right — perfect for tutorials, courses, and documentation layouts.

⌨ Keyboard Shortcuts
Del Delete selected Ctrl+Z Undo Ctrl+Y Redo Ctrl+D Duplicate Ctrl+E Export HTML ↑↓←→ Nudge 1px Shift+Arrow Nudge 10px Right-click Context menu
💡 Pro Tips
Right-click any element on the canvas for a context menu — Duplicate, Bring to Front, Send to Back, or Delete without opening any panel.
EgoTECH Nav component adds a real branded navbar matching egotechworld.com style — drop it at the top of your canvas for an instant EgoTECH-themed header.
Layers tab is essential when elements overlap — click the layer to select hidden elements, then use Front/Back to reorder them without moving anything manually.
Undo up to 50 steps — Ctrl+Z repeatedly to step back through your full design history. Your session is preserved until you refresh the page.
